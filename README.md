# * WEATHERLY *
### ~ cute pixel weather app ~

A dark pixel-art weather app built with Python + Tkinter. Deep purple night sky, twinkling stars, and weather icons that glow different colors depending on the forecast.

![License](https://img.shields.io/badge/License-MIT-c77dff)
![No Dependencies](https://img.shields.io/badge/dependencies-none-ffe66d)

---

## ✨ features

- 🌌 **permanent dark pixel theme** — deep purple background with animated twinkling stars
- 🎨 **weather-reactive accent colors** — the icon, temp, and description glow a different color for each condition
  - ☀️ clear sky → golden yellow
  - ☁️ cloudy → soft steel blue
  - 🌧️ rain → ocean blue
  - ⛈️ thunderstorm → electric purple
  - ❄️ snow → icy white-blue
  - 🌫️ mist → dusty lavender
- 🕹️ **pixel art sprites** that bounce for each weather condition
- ⌨️ **pixel-style GO! button** with a press-down effect
- 🌍 search any city in the world
- 🪶 **zero dependencies** - pure Python standard library, no pip needed!

## 🚀 getting started

### 1. clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/weatherly.git
cd weatherly
```

### 2. run it
```bash
python weather_app.py
```

> **Linux users:** if tkinter isn't installed:
> ```bash
> sudo apt-get install python3-tk
> ```

that's it! no API key, no pip install, nothing. just run and go 🎉

## 🌦️ weather conditions

| condition | accent color | sprite |
|-----------|-------------|--------|
| clear sky | `#ffe66d` golden | `* * *` sun rays |
| cloudy | `#b0c4de` steel blue | `) )` cloud puffs |
| rain | `#89cff0` ocean blue | `/ / /` rain lines |
| thunderstorm | `#cc99ff` electric purple | `/!/` lightning bolt |
| snow | `#cce8ff` icy blue | `* * *` snowflakes |
